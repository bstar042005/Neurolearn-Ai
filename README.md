## 📖 About the Project

NeuroLearn AI is an AI-powered platform designed to help identify early signs of dyslexia in school children and provide personalized learning support.

The goal of this project is to make dyslexia screening simple, accessible, and affordable for schools across India.

This project is currently being developed for a Hackathon.

---

## 🎯 Problem Statement

Many children with dyslexia remain undiagnosed because:

- Early screening is expensive
- Lack of trained specialists
- Schools have limited resources
- Delayed diagnosis affects learning

NeuroLearn AI aims to solve these challenges using AI-based assessments.

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/reocodes-51/NeuroLearn-AI.git
```

---

### 2. Move into the project folder

```bash
cd NeuroLearn-AI
```

---

## Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file

```
PORT=5000

MONGO_URI=your_mongodb_connection_string
```

Run backend

```bash
npm run dev
```

---

## Frontend Setup

Open another terminal

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on

```
http://localhost:5173
```

Backend runs on

```
http://localhost:5000
```
